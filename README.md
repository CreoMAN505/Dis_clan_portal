# 🏰 Clan Portal Bot

Discord бот для управления гильдией с системой регистрации, ролей и эвентов.

## 📋 Возможности

### 👤 Система регистрации
- Регистрация через красивое модальное окно
- Автоматическое изменение ника в Discord
- Разделение по классам (Воин, Маг, Хил, Лучник, Разбойник)
- Отслеживание Gear Score
- Иерархия ролей: Новичок → Рядовой → Ветеран → Офицер → Лидер

### 🎯 Система ролей и прав
- Жёсткая иерархия прав доступа
- Офицеры могут повышать/понижать игроков
- Проверка прав перед выполнением команд
- Автоматическая синхронизация с Discord ролями

### 📅 Система эвентов
- Создание эвентов (PvE, PvP, Raid, Донжон)
- Запись на эвенты с выбором роли:
  - 🛡️ Танк
  - 💚 Хил
  - ⚔️ DPS
  - ⚔️💚 DPS/Хил (гибрид)
  - 🛡️💚 Танк/Хил (гибрид)
- Ограничение по количеству участников
- Минимальный Gear Score для участия
- Автоматическое обновление состава
- Красивые кнопки для быстрой записи
- Детальная информация о составе группы

### 🛠️ Система билдов
- Добавление билдов (Ветеран и выше)
- Поиск билдов по классу
- Привязка билдов к профилю пользователя

### 📊 Активность
- Отслеживание времени в голосовых каналах
- Подсчёт сообщений
- Топ по онлайну

## 🚀 Установка

### 1. Требования
- Node.js v18+
- MongoDB Atlas аккаунт
- Discord аккаунт

### 2. Создание бота в Discord

1. Перейдите на [Discord Developer Portal](https://discord.com/developers/applications)
2. Создайте новое приложение ("New Application")
3. Перейдите в раздел "Bot" и создайте бота
4. **Важно:** Включите все три галочки в "Privileged Gateway Intents":
   - Presence Intent
   - Server Members Intent
   - Message Content Intent
5. Скопируйте токен бота

### 3. Настройка MongoDB

1. Зарегистрируйтесь на [MongoDB Atlas](https://www.mongodb.com/cloud/atlas)
2. Создайте бесплатный кластер (512MB)
3. В настройках кластера нажмите "Connect" → "Connect your application"
4. Скопируйте строку подключения (URI)

### 4. Установка бота

```bash
# Клонируйте репозиторий
git clone <your-repo-url>
cd clan_portal

# Установите зависимости
npm install

# Создайте файл .env
cp .env.example .env
```

### 5. Настройка переменных окружения

Отредактируйте файл `.env`:

```env
DISCORD_TOKEN=ваш_токен_бота
MONGODB_URI=ваша_строка_подключения_mongodb
CLIENT_ID=ваш_client_id
GUILD_ID=ваш_guild_id
PORT=3000
```

### 6. Получение IDs

**Client ID:** В Discord Developer Portal на странице вашего приложения в разделе "General Information"

**Guild ID:**
1. Включите Developer Mode в Discord (Settings → Advanced)
2. Правый клик по серверу → "Copy ID"

### 7. Запуск

```bash
# Развернуть команды (один раз)
npm run deploy-commands

# Запустить бота
npm start

# Или в режиме разработки
npm run dev
```

## 🎮 Команды

### Для всех:
- `/profile` - Посмотреть профиль (можно указать пользователя для просмотра чужого профиля)
- `/roster` - Список всех членов гильдии
- `/top_online` - Топ по времени в голосовых каналах
- `/build search` - Найти билды по классу
- `/event list` - Список всех эвентов
- `/event join` - Записаться на эвент (с выбором роли)
- `/event leave` - Покинуть эвент
- `/event_info` - Подробная информация об эвенте с составом группы

### Роли в эвентах:
При записи на эвент можно выбрать одну из ролей:
- 🛡️ **Танк** - Главный танкователь урона
- 💚 **Хил** - Лечитель группы
- ⚔️ **DPS** - Нанесение урона
- ⚔️💚 **DPS/Хил** - Гибрид: может наносить урон и лечить
- 🛡️💚 **Танк/Хил** - Гибрид: может танковать и лечить

### Для Ветеран и выше:
- `/build add` - Добавить новый билд
- `/event create` - Создать новый эвент

### Для Офицер и выше:
- `/promote` - Повысить пользователя
- `/demote` - Понизить пользователя
- `/profile @user` - Просмотреть чужой профиль

### Для Лидера:
- `/setup_portal` - Настроить портал регистрации

## 🌐 Развертывание на Render (бесплатно)

### 1. Подготовка

Загрузите код на GitHub.

### 2. Render.com

1. Зарегистрируйтесь на [Render.com](https://render.com)
2. Создайте "New Web Service"
3. Подключите ваш GitHub репозиторий
4. Настройки:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Environment Variables:** Добавьте все переменные из `.env`

### 3. UptimeRobot (чтобы бот не спал)

1. Зарегистрируйтесь на [UptimeRobot](https://uptimerobot.com)
2. Создайте новый монитор
3. Тип: HTTP
4. URL: ваша-ссылка-onrender.com
5. Interval: 14 минут

Теперь ваш бот будет работать 24/7 абсолютно бесплатно!


## 🎨 Иерархия ролей

1. **Новичок** - Новые участники гильдии
2. **Рядовой** - Полноценный член гильдии
3. **Ветеран** - Опытный игрок (может добавлять билды и создавать эвенты)
4. **Офицер** - Модератор (может повышать/понижать, смотреть профили)
5. **Лидер** - Глава гильдии (все права)
